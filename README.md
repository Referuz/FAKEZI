# FAKEZI

## Tienda en linea FAKEZI (Platzi Fake Store API - API pública)
Tienda funcional en linea con implementación de framework bootstrap
### Modulos
* Inicio de sesión
* Registro de usuario
* Página principal mostrando todos los productos de la API
* Carrito
* Compra
* Facturación
* Filtrado de productos en la vista principal y en el carrito
